Bəs SASS(SCSS) nədir?

Syntactically (Awesome Style Sheets) ifadəsinin qısaltması olan SASS ən tanınmış CSS ön prosessorlarından biridir. CSS-də olmayan xüsusiyyətləri istifadə edərək CSS-i daha sürətli və rahat istifadə etməyimizə kömək olur. SCSS isə Sassy Cascading Style Sheets mənasını verir. SCSS çox vaxt SASS olaraq adlandırılırdı, çünki yalnız aralarında sadəcə sintaktik fərqi var idi. Bildiyimiz kimi CSS-də bir class təyin etmək üçün {}, ; -bu simvollardan istifadə edirik. Ancaq SASS-da bu simvollardan istifadə etmək olmaz çünki SASS bu simvolları dəstəkləmir.

Parametrli SASS funksiyaları

Sass-da parametrləşdirilmiş funksiyalar bizə arqumentləri qəbul edən təkrar istifadə edilə bilən kod parçaları yaratmağa imkan verir. Buna dair aşağıda nümunə göstərilmişdir: 

            // Gradient arxa fon üçün parametrli Sass funksiyası
            @function generate-gradient($reng1, $reng2, $istigamet) {
            @return linear-gradient($istigamet, $reng1, $reng2);
            }

            // Funksiyanın istifadəsi
            .element {
            background: generate-gradient(#3498db, #e74c3c, to right);
            }
